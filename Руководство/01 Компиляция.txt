Для начала, у Вас должна быть установлена платформа .Net Framework 2.0, которую можно скачать с сайта Майкрософта. Также необходимо указать в системной переменной PATH путь к компилятору csc.exe. В Windows XP это можно сделать с помощью диалогового окна Переменные среды (Мой компьютер -> Свойства -> Дополнительно -> Переменные среды). Не забудьте перезагрузить компьютер, чтобы изменения вступили в силу. Путь к компилятору зависит от версии .Net Framework и может быть, к примеру, таким: C:\Windows\Microsoft.NET\Framework\v2.0.40607.

Распакуйте архив в отдельное место и запустите Компилировать.bat. Если в процессе компиляции возникнут ошибки, сообщения о них запишутся в файл Журнал.txt. Если ошибок не возникнет, у Вас появится файл Конструктор.exe, который уже можно запускать и использовать.
